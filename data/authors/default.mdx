# 關於我

我是一位前端開發，主要在電商平台、物流領域有實務經驗。從一開始的菜鳥工程師，到接觸北美大型電商網站的系統架構，再到物流系統的從 0 到 1 和團隊夥伴們架起一套現代化前後端分離的訂艙系統。

技術由早期的 JavaScript、jQuery，到使用 TypeScript、React、Next.js、React Native，另外也有涉略一些後端 service api 相關的實作，像是使用 C# .NET Core。

近幾年，在 AI 崛起的浪潮下，尤其是近期更聰明、更可靠的 AI 模型，我開始探索 AI 在軟體開發上的應用，並期望將這些經驗與想法記錄在我的部落格中。
